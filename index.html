NodeJS Microservices

Well-documented NodeJS microservices in HapiJS framework.

Constructed and built backend notification service, integrated with RabbitMQ to queue jobs triggered by other services, keep record on jobs running status and job payload. Also provides the feature to send out email/SMS/mobile app notification to notify clients. Constructed with HapiJS framework, implemented enqueue/dequeue feature by RabbitMQ, sending email feature by nodemailer, sending SMS feature by AWS SNS, pushing notification feature by Google Firebase. Also created a HapiJS plugin to call javascript APIs of Google Firebase.

  • Date: July 2016
  • Tech Stack: NodeJS, HapiJS, AWS, Google Firebase
  • Developer: Jixian Ma

Deep Defense Dashboard

MEAN stack project for online deep learning training/testing.

Deep defense project took advantage of the highly responsive feature of MEAN stack, created the tool to train/test DDoS attack from specific source, then present the results in dynamic chart. Integrated with Baidu Echarts library to generate dynamic chart and table based on deep learning results. All results are stored and fetched from MongoDB.

  • Date: Nov 2015
  • Tech Stack: MongoDB, NodeJS, Express, AngularJS
  • Developer: Jixian Ma, Menglei Sun
